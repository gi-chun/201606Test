  • 청구서를 다시 받고싶어요(분실/파손)
    먼저 당사 홈페이지 요금조회에서 청구금액 확인 및 안내된 ‘입금전용계좌’ / 홈페이지 ‘카드납부’로 납부가 가능하며, 해당 지역서비스센터에 방문하시면 청구서없이 신용카드 및 현금납부가 가능합니다. (지역서비스센터 찾기 : 당사 홈페이지 -> SERVICE -> 이삿날체크포인트 -> 맨하단 우리동네 서비스센터 찾기)

    * 홈페이지 청구금액 및 입금전용계좌 안내 : 요금조회 → 요금조회 → 로그인 / 납입자번호 클릭 → 조회기간 설정 → 청구금액 및 입금전용계좌 클릭 후 확인
    * 홈페이지 카드수납 : 코원에너지서비스 홈페이지 → 로그인 → 요금조회 클릭 → 요금조회 항목 클릭 → 요금카드수납(항목 하단)을 클릭하시고 실행하시면 됩니다.
    * 방문/홈페이지 결재 가능 신용카드 : 신한카드 / 하나SK카드 / 현대카드 / 씨티카드 / 삼성카드 / 롯데카드 (모든 카드 B.C계열 제외)

    상기 납부가 어려우신 고객님은 당사 콜센터(T1599-3366)로 청구서 재발급을 요청하시면 해당 지역 서비스센터에서 재발송하여 드립니다.
  • 도시가스요금 계산시 적용되는 보정계수/열량계수가 무엇인가요
    보정계수와 열량계수는 가스요금 산출시 적용되는 계수이며 아래와 같습니다.(청구서 상세내역내)

    - 보정계수 : 계량기에 의해 측정된 1㎥의 가스를 표준상태(0℃, 1기압) 부피로 환산하기 위한 계수를 말합니다.
    - 온압보정기 설치 수요처를 제외한 사용세대는 모두 보정계수를 적용하여 청구되고 있습니다.
    ※ 보정사용량(㎥) = 검침사용량(당월지침-전월지침(㎥)) x 보정계수
    - 한국표준과학연구원에서는 지역별, 계절별 사용량을 감안하여 온압보정계수 산출작업을 하여 매년 1월 1일자로 보정계수를 변동합니다.
    - 열량제도 : 도시가스요금을 사용한 부피(원/㎥)로 산정하던 것을 2012년 7월 1일부로 사용량을 열량계수(MJ/N㎥)를 적용 환산하여 열량(원/MJ)으로 요금을 납부하게 하는 제도입니다.
    - 2012년 7월 이전에는 도시가스열량을 10,400kal/N㎥으로 맞춰 공급하였으나, 이 후 정해진 범위 (10,100kcal/N㎥(41MJ/N㎥)) ~ 10,600kcal/N㎥(44.4MJ/N㎥))내에서 공급하고 있습니다.
    - 열량계수(평균열량) : 해당 기간 동안 한국가스공사 수급기지의 월간가중평균열량(MJ/N㎥)을 말합니다.
    ※ 청구량(MJ) = 보정사용량(㎥) x 열량계수(MJ/N㎥)

  • 도시가스요금 계산은 어떻게 하나요
    - 주 택 용: [(사용량 X 보정계수) X 평균열량 X 요금단가 + 기본료 + 교체비] + 부가세(10%)
    - 기타용도: [(사용량 X 보정계수) X 평균열량 X 요금단가 + 교체비] + 부가세(10%)

    - 보정계수란 기체로 공급되는 도시가스는 온도와 압력에 부피가 팽창/수축될 수 있으므로 지역별 평균 기온 등을 감안하여 온도와 압력차를 해소하기 위한 보정계수를 매년 산출 적용하는 계수
    - 평균열량이란 해당 기간의 월간가중평균열량(MJ/Nm³)
    - 주택용 중 취사+난방 겸용 세대는 516MJ까지 취사단가를 적용(경기지역만 해당)
    - 7등급 이하 가스계량기 교체비 없음
  • 가스요금이 연체되었습니다. 연체료 계산은 어떻게 하나요
    고객님께서 사용하신 도시가스 사용요금에 대해 납부기일 내 납부하지 못하신 경우 연체료가 부과됩니다.
    ☞ 연체료 산정 : 미납원금(부가세 미포함) 월2% 일할 계산하여 2회까지 부과, 익익월 청구서에 부과

    예시)
    - 1월 도시가스요금 : 10,000원, 납기일 10일 경과
    - 10,000원 x 0.02 x 10일(경과일수) / 31일(납기마감일의 당월 일수) = 60원(원단위 절사)
    - 따라서, 1월 연체료는 일할계산되어야 하기에 3월 청구서에 부과되어 연체료(60원)가 추가 청구됩니다.
  • 도시가스요금을 장기 체납을 하게 되면 어떻게 되나요
    - 가스요금 납부일 경과 후, 2회 이상 독촉장을 받고도 납부하지 아니한 경우에는 공급규정에 의거 공급중지 될 수 있습니다.
    - 체납으로 인하여 공급중지 되었을 경우 보일러 동파 및 기타 인적, 물적 피해에 대하여 당사에서 책임질 수 없음을 안내하오니 양지하시기 바랍니다.
    - 체납으로 인한 공급중지 후 재 연결 요청 시 소정의 해제수수료가 부과됩니다.
    - 가스공급 중지 후에도 미납 시에는 체납고객을 상대로 법적 조치를 할 수 있으며, 이로 인해 신용상의 불이익 을 당할 수 있습니다.
  • 도시가스 체납요금을 전부 납부하였습니다. 공급을 다시 언제 해주나요
    체납요금을 납부하신 후 기 통보된 신용정보사에 연락하시면 납부 확인 후 지역서비스센타 직원이 방문하여 도시가스를 공급하여 드립니다.
  • 사회취약계층(사회적배려대상자, 차상위계층, 다자녀)에 대한 도시가스요금 경감은 어떤 할인혜택이 있나요

    - 다 자 녀 가구 : 세대주의 자 또는 손이 각각 3인 이상인 가구

    구분 / 사회적배려대상자 / 차상위계층 / 다자녀가구
    취사용 / 1,680 / 840 / 420

    취사+ 난방 / 기타월 / 6,600 / 3,300 / 1,650

    * 동절기(12~3월), 기타월(4~11월)
    - 월 도시가스 사용요금이 경감금액보다 작은 경우 기본요금을 제외한 사용요금 만큼만 경감
    - 신청한 달의 다음달 사용기간부터 적용

    - 개인정보활용동의서(지역서비스센터내 비치)

  • 가스요금단가가 왜 이리 자주 바뀌나요(도시가스요금 연동제)

    - 정압기실 주변 침하가 발생되었거나 발생 증후가 보이는 경우

  • 검침원이 계량기 검침을 할 수 없어 인정고지를 했다고 하는데 무엇인가요? 계량기 고장시 사용량 산출근거는 무엇인가요?
    사용자의 부재로 검침을 할 수 없을때에는 전년 동월 사용량 또는 전월 사용량을, 계량기의 고장 등 그 밖의 사유로 사용량을 명확히 계측할 수 없을 때에는 전년 동월 포함 전후 1개월(3개월)간의 월평균사용량으로 산정하는 것을 원칙으로 하며, 부득이한 경우에는 전월사용량 또는 월사용 예정량을 기준으로 산정하여 통지 및 청구하는 제도입니다.
    (관련근거:서울시 공급규정 제4장 18조, 경기도 공급규정 제 4장20조 사용열량의 산정)
  • 도시가스 연결비용을 알고싶어요
    서비스 비용 안내는 자재비등으로 인해 고객센터별로 다소 차이가 있을 수 있으며, 당사 홈페이지 SERVICE -> 가스와 안전 -> 이삿날 체크포인트 내용중 서비스 항목별 수수료 내역을 참고하시면 됩니다.

    - 휴즈콕이나 가스호스를 가져오신 경우 해당 자재비를 절감할 수 있습니다. (자재의 노후에 따라 교체 필요시 고객님께서 반드시 확인 후 교체)
    - 기타 세부사항은 당사 콜센터(T1599-3366)로 연락주시면 상세히 설명드리겠습니다.
  • 이사시 도시가스요금 정산은 어떻게 하나요?
    이사 가시기 2~3일 전에 당사 콜센터(T1599-3366)에 전화로 사전 예약을 하시면 관할 지역 서비스센터 직원이 방문하여 고객님의 계량기 사용량을 확인한 후 정산을 하여드립니다.
    - 이사 정산 요금은 현금 or 카드 납부가 가능합니다.
    * 신한카드 / 하나SK카드 / 현대카드 / 씨티카드 / 삼성카드 / 롯데카드 (모든 카드 B.C계열 제외)
    - 고객님의 이사에 따른 가스레인지 철거는 별도의 비용을 청구하지 않습니다.
  • 이사 가려 합니다. 어떻게 해야 하나요?
    이사(전출) 가시기 2~3일 전에 당사 콜센터(T1599-3366)에 전화로 이사(전출)일시, 주소, 연락처 등을 알려주시면 해당일자에 관할 지역서비스센터 직원이 방문하여 연소기와 연결된 호스와 중간밸브를 철거하고 안전조치를 합니다.

    - 가스레인지를 철거하지 않고(빌트인 세대) 이사를 가시더라도 반드시 사전에 요금청구서에 인쇄된 서비스 센터 전화번호로 연락하시어 안전조치를 받으셔야 합니다.
    - 이사(전출) 당일 지역서비스센터 직원이 도시가스 호스 철거 작업 시 가스계량기 지침을 확인 후 도시가스 요금을 정산합니다.
    - 자동이체로 요금을 납부하시는 고객님께서는 이사 전 미리 자동이체를 해지하여 주시기 바랍니다.
    - 고객님의 이사에 따른 가스레인지 철거는 별도의 비용을 청구하지 않습니다.
    - 전출 후 도시가스요금 할인(경감) 대상 자격이 유지되는 경우 아래와 같은 장소로 신청하셔야 합니다.
    * 당사 전입 → 가까운 지역서비스센터 (지역서비스센터 찾기 : 당사 홈페이지 -> SERVICE -> 이삿날체크포인트 -> 맨하단 우리동네 서비스센터 찾기)
    * 타사 전입 → 타사 경감 접수 장소
  • 이사 왔습니다. 어떻게 해야하나요?
    이사 오시기 2~3일 전에 당사 콜센터(T1599-3366)에 전화로 사전 예약을 통한 가스레인지 연결 예약을 하시면 해당일자에 관할 지역서비스센터 직원이 방문하여 가스레인지와 중간밸브를 연결하고 안전점검 후 도시가스를 공급합니다.

    * 가스레인지 연결은 반드시 “가스시설시공업(제2종)”이상의 면허를 보유한 전문 시공자에게 의뢰하여야 합니다. (관할 구청에 문의 또는 건설산업지식정보 시스템에서 조회가능)
    * 당사 서비스센터는 가스시설시공업(제2종) 이상을 보유하고 있습니다.

    - 도시가스 연결시 고객님과 당사간 도시가스사용 계약을 체결하고 있으니 협조하여 주시기 바랍니다.

    - 사용자 이사(전입)시 서비스 항목에 따른 소정의 수수료가 발생합니다.(도시가스 연결비용 안내 링크)

    - 도시가스요금 할인(경감) 대상 자격이 유지되는 경우 전입하시는 곳의 가까운 지역서비스센터에 재신청을 하셔합니다.(지역서비스센터 찾기 : 당사 홈페이지 -> SERVICE -> 이삿날체크포인트 -> 맨하단 우리동네 서비스센터 찾기)

    - 이사 당일 연결 신청하실 경우 예약고객부터 서비스가 우선 진행되오니 이점 양해하여 주시기 바랍니다.
  • 계량기 교체는 왜 하나요?
    가스계량기 교체는 "계량에 관한 법률"에 의해 아래와 같이 실시합니다.

    * 교체주기
    - 10등 이하 : 1회/5년
    - 16등 이상 : 1회/8년

    * 교체비용
    - 7등 이하 : 별도 교체비 없음 (기본요금 징수)
    - 10등 이상 : 월 분납 (공급규정에 의거 비용 산정)
  • 계량기에서 이상한 소리가 나네요. 도시가스를 사용하고 있는데 사용량 없이 청구되고 있았네요
    - 가스계량기에서 소음 등 이상 발견시 당사 콜센터(T1599-3366)로 연락주시면 해당 지역서비스 센터에서 계량기를 점검하여 고장이 확인 되면 교체하여 드립니다.
    - 도시가스를 사용하고 있는데 기본료 or 계량기 교체비만 청구되는 경우는 계량기 고장중 하나인 계량기 숫자판 고장(불회)일 가능성이 높습니다.
    - 불회는 가스계량기를 통해 가스는 통입되고 있으나 계량기 숫자판 고장으로 계량이 안되는 상태 를 말합니다.
    - 따라서, 고객님께서 상기와 같이 계량기 고장이라고 판단된다면 당사 콜센터(T1599-3366)로 연락주시기 바랍니다.
    ※ 불회로 확인될 경우 미정산된 요금은 서울시/경기도 공급규정 제4장에 의거 전년 동월 포함 전후 1개월(3개월)간의 월평균사용량으로 산정하며 부득이한 경우 전월사용량 또는 월사용예정량을 기준으로 산정합니다.
    - 고객께서 가스계량기의 계량이 의심될 경우 성능검사를 요청할 수 있으며, “계량에 관한 법률”에 에서 정한 사용공차를 초과 할때는 당사의 비용으로 신속하게 교체합니다.
    - 또한, 서울시/경기도 공급규정에 의거 외부기관에 계량기 성능검사를 의뢰할 수 있으며 이 에 소요 되는 비용은 고객이 부담합니다. 다만, 검사결과 사용공차를 초과할 때 에는 당사의 비용으로 합니다.
  • 가스누출경보기가 작동해서 경보음이 울려요.
    가스누출경보기의 경보음이 울리면 우선 계량기와 중간밸브를 Close(계량기, 중간밸브) 하시고 실내 환기 후 당사 상황실(T02-3410-8119) or 콜센터(T1599-3366)로 연락을 주시면 당사 직원이 출동하여 확인/점검하여 드립니다.

    확인 후 가스시설에 이상이 없고 경보기 오작동으로 확인이 되면 해당 제조사에 A/S 요청을 하셔야 합니다.(당사 홈페이지 경보기업체 링크)
    * 안내한 경보기 업체는 명단은 참고 자료로서 당사와 무관함을 알려드립니다.
  • 가스렌지가 이상해요
    가스렌지 주변에서 가스냄새가 나요-
    → 우선 계량기와 중간밸브를 Close(계량기, 중간밸브) 하시고 실내 환기 후 당사 상황실(02-3410-8119) or 콜센터(T1599-3366)로 연락을 주시면 당사 직원이 출동하여 확인/점검하여 드립니다.

    가스렌지 불꽃 점화가 안되요
    → 가스밸브 open(손잡이 방향이 배관과 일치) 되었는지 확인
    → 가스 공급 정상이 확인되고 불꽃점화가 안된다면 가스렌지 건전지 교체 (대부분 가스렌지 후면에 건전지 교체 캡이 있음)
    → 가스렌지 노즐 불량 : 가스렌지 청소 후 노즐 안착 불량(노즐 재안착) , 노후로 노즐 막힘 등의 상태 불량 등(가스렌지 제조사 A/S)
    → 밸브 open이 확인 후 가스 미공급이 확인될 경우 당사 콜센터(T1599-3366)로 연락주시기 바랍니다.

    가스렌지 불꽃이 주황색으로 전과(청색) 달라요-
    → 가스렌지 주변 가습기나 수분 분무기가 사용하고 있는지 점검(사용이 확인되면 일정거리 유지 후 확인)
    → 실내 환기 불량으로 실내 환기 실시
    → 가스렌지 노즐 불량 : 가스렌지 청소 후 노즐 안착 불량, 노후로 노즐 막힘 등의 상태 불량 등 (가스렌지 제조사 A/S)
    → 가스 정상공급이 아닌 것으로 의심되는 경우 당사 콜센터(T1599-3366)로 연락주시기 바랍니다.

    * 가스렌지 몸체에 제조사 A/S 전화번호가 안내되어 있습니다.
  • 보일러가 이상해요
    보일러주변에서 가스냄새가 나요
    → 우선 계량기와 중간밸브를 Close(계량기, 중간밸브) 하시고 실내 환기 후 당사 상황실(02-3410-8119) or 콜센터(T1599-3366)로 연락을 주시면 당사 직원이 출동하여 확인/점검하여 드립니다.

    보일러가 정상운전을 안해요
    → 밸브 open 확인(계량기, 보일러 차단밸브)
    → 보일러 상태 점검(보일러 자체 이상 확인 - 보일러 A/S 요청)
    → 중간밸브 open 및 가스렌지 점화로 가스공급 확인 - 미점화시 당사 콜센터(T1599-3666)로 연락

    * 가스보일러 불편신고 전화번호 링크
    * 가스보일러 사용 도중 불편사항이나 A/S요청은 고객님 댁의 보일러 제조회사를 확인 후 안내한 해당 회사에 연락하여 조치 받으시기 바랍니다.
    * 안내한 보일러 업체 명단은 참고 자료로서 당사와 무관함을 알려드립니다.
